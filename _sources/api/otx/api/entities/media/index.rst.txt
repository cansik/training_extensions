.. py:class:: IMediaEntity

   Media entity interface.

   This interface is used to represent any kind of media data, on which users can annotate and tasks can perform
   training/analysis.


.. py:class:: IMedia2DEntity

   Bases: :py:obj:`IMediaEntity`
