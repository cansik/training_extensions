abstract forward(image: Any) → Any[source]

Forward the input image to the model and return the output.

NOTE: The input is typed as Any at the moment, mainly because it could be numpy

array,torch Tensor or tf Tensor. In the future, it could be an idea to be more specific.

This method should perform the prediction by forward-passing the input image to the model, and return the predictions in a dictionary format.

For instance, for a segmentation task, the predictions could be {“mask”: mask}.

class otx.api.usecases.exportable_code.inference.inference.BaseOpenVINOInferencer(model_file: Union[str, bytes], weights_file: Union[str, bytes, None] = None, device: str = 'CPU', num_requests: int = 1)[source]

Bases: BaseInferencer, abc.ABC

Base class for OpenVINO inference.

Can handle the basic flow of reading and loading a model. If the network needs to be reshaped, override the load_model function. One would need to implement the following methods to use this as OpenVINO Inferencer

  • pre_process

  • forward

  • post_process

Parameters

  • weight_path – Path to the weight file

  • device – Device to use for inference. Check available devices with IECore().available_devices.

  • num_requests – Number of simultaneous requests that can be issued to the model, has no effect on synchronous execution.

Raises

ValueError – Raised if the device is not available.

This class runs asynchronous inference on a BaseOpenVinoInferencer.

Using a BaseStreamer as input

Parameters

  • streamer – A streamer that provides input for the inferencer

  • inferencer – The inferencer to use to generate predictions

  • drop_output – Set to a number to limit the amount of results stored at a time. If inference is completed but there is no room for the output. The output will be dropped.Set to 0 to disable, Set to None to automatically determine a good value

__iter__() → Iterator[Tuple[numpy.ndarray, List[numpy.ndarray]]][source]

Starts the asynchronous inference loop.

Example

>>> streamer = VideoStreamer("../demo.mp4")
>>> inferencer = ExampleOpenVINOInferencer(weights="model.bin", num_requests=4)
>>> async_task = AsyncOpenVINOTask(streamer, inferencer)
>>> for image, predictions in async_task:
...    # Do something with predictions
