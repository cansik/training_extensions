convert_to_annotation(predictions: numpy.ndarray, metadata: Optional[Dict] = None) → otx.api.entities.annotation.AnnotationSceneEntity[source]

Converts a set of predictions into an AnnotationScene object.

Parameters

  • predictions (np.ndarray) –

    Prediction with shape [num_predictions, 6] or

    [num_predictions, 7]

    Supported detection formats are

    • [label, confidence, x1, y1, x2, y2]

    • [_, label, confidence, x1, y1, x2, y2]

    label can be any integer that can be mapped to self.labels confidence should be a value between 0 and 1 x1, x2, y1 and y2 are expected to be normalized.

  • metadata (Optional[Dict]) – (Unused)

Returns

AnnotationScene Object containing the boxes obtained from the prediction.

Return type

AnnotationScene
