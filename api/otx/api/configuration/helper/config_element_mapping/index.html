otx.api.configuration.helper.config_element_mapping

This module contains mappings from ConfigElementType names to the appropriate constructor classes or functions.

Currently three different mappings are defined: PrimitiveElementMapping representing the different base parameter types, GroupElementMapping representing the different configuration groups and RuleElementMapping representing the different ui exposure logic classes
