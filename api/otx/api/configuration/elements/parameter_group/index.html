class otx.api.configuration.elements.parameter_group.ParameterGroup[source]

A group of configuration elements.

Parameters living within the parameter group are typed attrs Attributes. The schema for each parameter is defined in its metadata, which can be retrieved using the get_metadata method from the parent ParameterGroup instance.
